Trivanox Web Admin

Trivanox Web Admin adalah aplikasi manajemen Human Resource (HR) berbasis web yang digunakan untuk mengelola data karyawan, absensi, izin & cuti, serta monitoring aktivitas karyawan secara terpusat. Aplikasi ini berperan sebagai panel administrator yang terintegrasi dengan aplikasi mobile Trivanox.

Repository Trivanox Mobile : https://github.com/ahmadzipur/Trivanox.git

Demo Aplikasi : https://ryzola.com/trivanox/
